A personal portfolio website showcasing my skills, career, and projects.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Animation**: Framer Motion
- **Language**: TypeScript

## Getting Started

```bash
git clone https://github.com/JaminYip/portfolio.git
cd portfolio
npm install
npm run dev
```
